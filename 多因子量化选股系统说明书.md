# 多因子量化选股系统说明书

## 一、系统概述

本系统是一套基于真实A股市场数据的多因子量化选股回测平台，面向个人投资者和量化研究者，提供从数据获取、因子计算、策略回测到绩效分析的完整工作流。

系统使用 Python 语言开发，数据来源为新浪财经、腾讯财经和 baostock 等公开数据接口，覆盖 2015 年 1 月至 2025 年 6 月共约 10 年的 A 股日线行情数据，涉及 3345 只在市股票。

### 1.1 设计目标

- **真实数据**：所有回测均基于真实市场行情，不使用模拟或随机生成数据
- **完整因子库**：涵盖动量、估值、质量、波动率、流动性五大类共 40 个因子
- **可复现性**：全部代码开源，数据获取方式公开，任何人可复现回测结果
- **实用性**：支持增量数据更新、多频率调仓、交易成本模拟等实际需求

### 1.2 适用场景

- 量化投资策略研究与验证
- 因子有效性检验与筛选
- 学术研究中的实证分析
- 投资组合构建的参考工具

## 二、数据体系

### 2.1 数据来源

| 数据类型 | 来源 | 更新频率 | 覆盖范围 |
|---------|------|---------|---------|
| 日线行情 | 新浪/腾讯/baostock | 日频 | 全A股（剔除ST） |
| 财务指标 | 东方财富 | 季频 | ROE/ROA/利润率等 |
| 股票列表 | 新浪实时行情 | 实时 | 代码/名称/成交量 |

### 2.2 数据规模

- 股票数量：3345 只（沪深主板 + 创业板）
- 时间跨度：2015-01-05 至 2025-06-20
- 月度样本：125 个月
- 数据字段：日期、开盘价、最高价、最低价、收盘价、成交量、成交额

### 2.3 数据质量控制

- 剔除 ST、\*ST 及退市整理股票
- 剔除上市不满 60 个交易日的次新股
- 剔除停牌期间的交易数据
- 对极端收益率进行裁剪处理（月收益率限制在 -99% 至 1000%）

## 三、因子体系

### 3.1 因子分类

系统构建了五大类共 40 个因子：

**第一类：动量因子（12 个）**

动量因子捕捉股价的趋势延续和反转效应。包括 1 个月、3 个月、6 个月、12 个月的区间收益率，短期反转因子（1 个月与 12 个月收益差），20 日和 60 日相对强弱指标（RSI），以及波动调整动量和信息离散度等。

**第二类：估值因子（8 个）**

基于财务数据计算的市盈率（PE）、市净率（PB）及其倒数（EP、BP），以及行业相对估值指标。估值因子衡量股票价格相对于基本面的偏离程度。

**第三类：质量因子（10 个）**

包括净资产收益率（ROE）、总资产收益率（ROA）、毛利率、净利率、流动比率、资产负债率、经营现金流与净利润比率、营收增长率、净利润增长率和现金流质量。这些因子反映企业的盈利能力和财务健康状况。

**第四类：波动率因子（6 个）**

包括 20 日和 60 日历史波动率、最大回撤、下行波动率和特质波动率。低波动异象在 A 股市场长期存在，低波动股票往往表现优于高波动股票。

**第五类：流动性因子（4 个）**

包括 20 日平均成交量、Amihud 非流动性指标、成交量波动率和换手率变化。流动性因子在 A 股市场具有显著的选股能力。

### 3.2 因子计算方法

每个因子按月度频率计算，采用截面标准化（z-score）处理。对于财务数据类因子，使用最新可得的季报数据（考虑报告期与实际披露期的滞后）。

行业中性化处理采用截面标准化方式：将每个因子值减去截面均值，再除以截面标准差，消除市场整体波动对因子的影响。

## 四、回测框架

### 4.1 框架设计

回测框架采用事件驱动架构，核心模块包括：

- **DataLoader**：数据加载器，提供 current() 和 history() 接口，支持按日期和字段查询
- **Position**：持仓管理，记录每只股票的股数、成本价和当前市值
- **Portfolio**：组合管理，维护总资产、现金余额和持仓列表，支持按股数下单和按目标比例调仓
- **BacktestEngine**：回测引擎，支持日频和月频调仓，内置佣金和滑点模型

### 4.2 交易成本假设

| 项目 | 费率 |
|------|------|
| 佣金（双边） | 0.15% |
| 滑点 | 0.10% |
| 单次交易总成本 | 约 0.35% |

### 4.3 调仓机制

采用月度调仓方式，每月最后一个交易日执行调仓。调仓时根据因子值对全市场股票排序，按五分位分组，等权持有多头组合。

## 五、策略绩效

### 5.1 因子有效性检验

通过 Rank IC（斯皮尔曼秩相关系数）检验因子与下月收益的相关性。有效因子应具备：

- IC 均值绝对值大于 0.03
- IR（IC 均值 / IC 标准差）绝对值大于 0.3
- IC 大于 0 的月份占比超过 50%
- t 统计量绝对值大于 1.96（95% 置信水平）

**核心因子 IC 统计：**

| 因子 | IC 均值 | IR | IC>0 占比 | t 统量 | 含义 |
|------|--------|-----|----------|--------|------|
| amihud_20d | 0.065 | 0.440 | 65.3% | 4.90 | 低流动性溢价 |
| volatility_20d | -0.073 | -0.447 | 31.4% | -4.98 | 低波动异象 |
| volume_vol_20d | -0.046 | -0.537 | 28.0% | -6.00 | 成交量稳定 |
| mom_20d | -0.059 | -0.431 | 31.5% | -4.79 | 短期反转 |
| ret_1m | -0.056 | -0.424 | 34.7% | -4.72 | 近1月反转 |

### 5.2 Amihud 因子分层回测

以 Amihud 非流动性因子为例，将全市场股票按因子值从低到高分为 5 组，等权持有并月度调仓：

| 组合 | 累计收益 | 年化收益 | 夏普比率 | 最大回撤 | 月度胜率 |
|------|---------|---------|---------|---------|---------|
| G1（高流动性） | 287.2% | 13.9% | 0.449 | -35.3% | 54.4% |
| G2 | - | - | - | - | - |
| G3（中间） | 70.6% | 5.3% | 0.097 | -58.5% | 49.6% |
| G4 | - | - | - | - | - |
| G5（低流动性） | 947.8% | 25.3% | 0.581 | -41.4% | 56.8% |
| 多空组合（G5-G1） | 213.5% | 11.6% | 0.369 | -43.0% | 49.6% |

G5 组（低流动性股票）年化收益 25.3%，显著优于 G1 组（高流动性股票）的 13.9%，验证了 A 股市场低流动性溢价的存在。

### 5.3 复合因子策略

基于 IC_IR 加权方式合成复合因子，筛选标准为 IR 绝对值大于 0.1。最终入选因子及其权重：

- amihud_20d（77.2%）：Amihud 非流动性指标
- max_drawdown_60d（22.8%）：60 日最大回撤

复合因子 G5 组年化收益 22.5%，夏普比率 0.636，表现出色。

## 六、风险提示

1. **历史回测不代表未来表现**。市场环境变化可能导致因子失效。
2. **交易成本影响**。实际交易中佣金、印花税和冲击成本可能高于假设值。
3. **数据偏差**。回测使用前复权数据，可能存在幸存者偏差。
4. **流动性限制**。小盘低流动性股票在实际交易中可能面临买卖价差大、难以成交等问题。
5. **因子拥挤**。当大量资金追逐同一因子时，因子收益可能衰减。

## 七、使用说明

### 7.1 环境要求

- Python 3.10 或更高版本
- 依赖库见 requirements.txt

### 7.2 运行步骤

```
pip install -r requirements.txt
python 01_数据获取与存储.py      # 获取数据（约1小时）
python 02_财务数据与因子计算.py   # 计算因子
python 03_回测引擎搭建.py        # 运行回测
python 04_多因子选股实跑.py      # 多因子分析
python 05_策略优化与交付.py      # 优化与报告
```

### 7.3 输出说明

- `data/daily/`：日线行情 CSV 文件
- `data/factors/`：因子值 CSV 文件
- `output/figures/`：可视化图表
- `output/*.csv`：绩效统计表
